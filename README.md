# Multi-Timer

一个功能强大的 Windows 多定时器/闹钟桌面应用程序，支持同时管理多个定时任务。

![Multi-Timer 主界面预览](ImgAuthor/WinMain.png) <!-- 如果有截图可替换 -->

## 特性

- **同时支持 8 个独立定时器**
- **两种定时模式**：
  - **闹钟模式**（指定具体时间触发）
  - **倒计时模式**（从当前时间开始倒计时）
- **实时模拟时钟显示**（可切换显示列表/仅时钟模式）
- **状态可视化**：
  - 未配置（灰色）
  - 已开启（黑色）
  - 激活中（红色）
- **批量操作**：一键全部开启/关闭/清除
- **保存与加载**：支持将定时器配置保存为 XML 文件并加载
- **精确定时**：高精度定时更新
- **简洁直观的 Windows Forms 界面**
